# State Machine LED Controller

This project implements a simple **state machine** using a push button and RGB LED on the **Arduino Nano 33 BLE Sense**.

###  System Overview
- External button on pin D4 triggers state transitions
- Built-in RGB LED displays current state:
  - Red → State 1
  - Green → State 2
  - Blue → State 3
- Demonstrates finite-state logic and debounced GPIO input handling

###  Learning Outcomes
- Finite state machine design
- Interrupt-driven input handling
- Real-time embedded LED control
